# Project setup
## Steps:
### 1) Clone repository
    git clone https://github.com/prajapatiomkar/time-zone-calendar-scheduler.git
### 2) Move into project
    cd time-zone-calendar-scheduler
### 3) Install dependency
    npm install
### 4) Run the project
    npm run dev
